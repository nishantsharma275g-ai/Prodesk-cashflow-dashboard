# 💰 Prodesk Cash Flow Dashboard

A modern and interactive **Cash Flow Dashboard** built using **HTML, CSS, and Vanilla JavaScript**. This application helps users manage their personal finances by tracking salary, recording expenses, monitoring remaining balance, visualizing spending with charts, and generating downloadable reports.

---

## 🚀 Live Demo

https://prodesk-cashflow-dashboard.vercel.app/
---

## ✨ Features

### Phase 1 – Base MVP

- Add Total Salary
- Add Expense Name
- Add Expense Amount
- Real-time Remaining Balance Calculation
- Dynamic Expense List
- Form Validation
- Built completely with Vanilla JavaScript

---

### Phase 2 – Data Persistence & Visualization

- LocalStorage Support
- Data persists after browser refresh
- Delete Individual Expenses
- Dynamic Pie Chart using Chart.js

---

### Phase 3 – Advanced Features

- PDF Report Generation using jsPDF
- Low Balance Warning Banner
- Warning Sound Alert
- Live Currency Conversion (INR, USD, EUR, GBP)
- Exchange Rate API Integration

---

## 📊 Project Workflow

1. User enters salary.
2. User adds expenses.
3. Dashboard updates automatically.
4. Remaining balance is calculated.
5. Expense data is stored in LocalStorage.
6. Pie chart updates dynamically.
7. User can delete expenses.
8. PDF report can be downloaded.
9. Currency conversion updates values using a live API.
